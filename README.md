# solidjs-test

SolidJS の技術調査を兼ねて簡単なデモアプリを作成。

開発環境: Vite + SolidJS + TypeScript

[SolidJS 公式のテンプレート](https://www.solidjs.com/guides/getting-started)

# Development

1. 依存関係のインストール

```properties
yarn install
```

2. 開発サーバー立ち上げ

```properties
yarn dev
```

3. ブラウザでアクセス

ブラウザで [http://localhost:3000](http://localhost:3000) にアクセスして開発する。
